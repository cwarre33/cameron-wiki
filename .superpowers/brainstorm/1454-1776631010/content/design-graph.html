Design — Section 2: Graph Model & D3 Visualization

Four node types, three edge types, one self-contained HTML output. Does this look right?

Graph schema (NetworkX → GraphML)
Node types
ip — one node per exposed host. Attrs: org, city, vuln_count, port
asn — one node per autonomous system. Attrs: asn_number, org_name, ip_count
country — one node per country code. Attrs: name, total_exposed
protocol — 4 nodes: modbus, bacnet, dnp3, s7. Sized by total exposure count
Edge types
ip → protocol  exposes
ip → asn  belongs_to
asn → country  located_in
D3 output (self-contained HTML)
modbus bacnet dnp3 s7 all

Single .html file, no server needed. Drag nodes, hover for tooltip, click protocol pills to filter. Saved to raw/osint/YYYY-MM-DD-viz.html.
